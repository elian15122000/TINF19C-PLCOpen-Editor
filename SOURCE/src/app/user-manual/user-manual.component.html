User Manual

Navigator

There is a bar at the top of each (sub) page. On it there is a house icon for navigation back to the homepage. Next to the home button is an about button, which you can use to get to the about page, unless you are in the editor. In addition, you can always see the name of the project or product including version (currently PLCOpen-Editor 1.0) on the bar. On the homepage you can click on three boxes. Through them you can create a new project, upload and open an existing project or access the user manual. If you decide to create a new project an input opens, where you can enter the project name and open it. If you click the button to open an existing project, you can upload a file to open it. If a project is opened, you can see the overview with the different POUs (programs). There you can add a new POU, delete an existing pou, or export your project by clicking the button 'export'. Furthermore you can change to another project by clicking the button 'Open another project'. If you decide to add a new POU, you will be asked to enter a name, the type and the language of the POU. For a language you can choose between Sequential Function Chart(SFC) and Function Block Diagram (FBD). For a type you can choose between programm, function and function block. To open a POU and therefore get to the editor you simply have to click on the name of the POU.

Converter

Import

Click on 'Open Project' on the homepage to upload an existing file. Choose the file and click 'open'. By clicking this button the file gets imported. The pous get read and appear in the pou list. Click then on a specific pou to load the pou and illustrate it on the editor.

Export

While editing you can press the button with the save icon to save your current version to the xml file. To export the xml file you can go back to the pou overview, where there is a export button. By clicking the button, the xml-file of the project gets exported.

Editor

Adding a function block

On the right side in the library choose a function block you want to add. By clicking 'add to the editor' it will appear on the left side in the editor. By using drag and drop you can move the position of the function block within the editor.

Adding variables

Click the '+'-button to add a new variable. There you can set a name, a class, an IEC-address, an initial value, options and you may add a documentation. You can also delete variables by clicking the delete icon. To add a variable to the editor, you can click the button 'add variable', where you can choose the variable you want to add, and you can choose the type of the variable and if it shall be negated. By clicking the button 'add' the variable will appear on the editor.

Adding connections

Click on the element, you want to add a connection to. There will appear dropdown lists, where you can choose what the input and output of a variable shall be connected with.
